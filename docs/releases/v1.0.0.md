# v1.0.0 发布说明

这是 `copilot-help` 的首个正式发布版本。

## 发布目标

发布一个可用的中文 GitHub Copilot 实战模板库 `v1.0.0`，核心目标是：

- 给 VS Code 用户提供一套可复用的 Copilot 自定义资产
- 覆盖中大型项目中的高频工作流
- 让仓库具备真实项目接入和后续反馈迭代的基础能力

## 本次发布包含什么

### 教程与说明

- 大型项目使用教程
- VS Code 使用说明
- 真实项目接入指南
- 当前版本说明

### Copilot 自定义资产

- 14 个 prompts
- 4 个 agents
- 2 个 skills
- 2 个文件级 instructions
- 1 个项目级 instructions 文件

### 公开发布基础资产

- README 导航
- LICENSE
- CHANGELOG
- CONTRIBUTING

## 当前版本定位

`v1.0.0` 的定位是“可发布、可试用、可接入真实项目的第一版”，不是穷尽所有场景的最终版。

## 推荐使用方式

如果你第一次使用，建议从最小组合开始：

1. `feature-plan`
2. `code-review`
3. `bugfix`
4. `pr-description`
5. `planner`

## 已知边界

- 当前模板更偏通用工程实践，还没有按具体技术栈细分
- 尚未基于大量真实团队反馈做二次收敛
- 后续版本建议按真实项目验证结果小步迭代

## 后续版本建议

后续不建议再无目标扩充文件数量，而应重点关注：

- 哪些模板真实高频使用
- 哪些模板命中率不高
- 哪些角色或工作流需要精简或增强
